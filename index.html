<notification-elements>

Provides Notifications very easy using Web notification API and web components.

The first demo, we have some standard Notification attributes, which when clicked triggers the notification

Simple notification

<notification-elements header="Bienvenido" body="This is a simple notification!"></notification-elements>

The second demo, you can define which icon, title, body, language, and Duration using attributes of notification, and other things.

<notification-elements id="button-notification-1 header="Meooow!" body="I'm the Octocat!" icon="http://forum.webflow.com/plugins/emoji/images/octocat.png"></notification-elements>

<script>
var demoBtn = document.querySelector('#demo-btn'),
    notifyElement = document.querySelector('#button-notification-1');

demoBtn.addEventListener('click', function(e) {
    notifyElement.notify();
});
</script>

Attributes

Attributes Options Default Description
icon String icon.png Specifies whether there is a icon in the notification.
body String Bienvenido Specifies your body notification
header String This is a simple notification! Specifies your text notification
duration int 10 Specifies the time of notification

Events

Event Description
onclick It is triggered each time the user clicks on the notification.
onshow It is triggered when the notification is displayed.
onerror It is triggered each time the notification encounters an error.
onclose It is triggered when the user closes the notification.

Usage

1. Import Web Components Polyfill

<script src="bower_components/platform/platform.js"></script>

2. Import Custom Element

<link rel="import" href="src/notification-elements.html">

3. Using It

<notification-elements></notification-elements>

4. Enjoy it!
